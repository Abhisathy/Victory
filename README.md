## Getting Started
To get started with the development of dashboard:

- Clone the Project

- Create a Virtualenv:   
` virtualenv venv` 

- To Create and Activate Virtualenv:  
    - In Windows:   
        `virtualenv venv`   
        `venv\Scripts\activate`   

    - In Linux:  
        `virtualenv venv`  
        `source venv/bin/activate`  

- To Install all requirements:  
` pip install -r requirements.txt`
